## Features
- **Supports Format Specifiers**:
  - `%d`: Prints integers.
  - `%f`: Prints floating-point numbers.
  - `%s`: Prints strings.
  - `%c`: Prints single characters.
- Outputs the formatted string to the standard output (`STDOUT`) using the `write()` system call.
