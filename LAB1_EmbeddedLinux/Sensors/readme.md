# Sensor Data Processing Project

## Run the Project:

- Ensure the log files exist:
    Create logs/temperature.log and logs/pressure.log with the appropriate format.

- Make the script executable:

```bash
    chmod +x run_project.sh 
```
- Run the script:
```bash
Copy
./run_project.sh
```
- Example Output;
```Plaintext
Running the program...
Temperature (Kelvin): 176.11 K
Temperature (Celsius): 80.06 C
Pressure (hPa): 1070.92 hPa
Pressure (Pa): 107092.00 Pa
```
